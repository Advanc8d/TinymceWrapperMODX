FILE MANAGER

I tried using your MODx native File Manager, and my brain stopped computing. So instead, I am using the awesome Responsive Filemanager, which you can customize all you want.

  1. Go and rename this file (remove the suffix) [[++assets_url]]components/tinymcewrapper/responsivefilemanager/filemanager/config/config.php.modxReady
  2. Go and enter the path to your core folder in this file [[++assets_url]]components/tinymcewrapper/responsiveMODxConfig.php

If you want to change the root upload folder, go ahead, only make sure you change the other few relative references.
