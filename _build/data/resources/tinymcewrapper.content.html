FILE MANAGER

I tried using your MODx native File Manager, and my brain stopped computing. So instead, I am using the awesome Responsive Filemanager, which you can customize all you want.

  1. RFM should already be unzipped for you. Re-Unzip whenever you run into chaotic troubles.
    If ever you do, please Copy all files in rfmConfigForModx-backup to filemanager/config/
  2. Enter in the path to your MODx site's index.php to filemanager/config/config.modx.php

This procedure is to demonstrate how independent the Responsive FileManager is from MODx or TinyMCE.

If you want to change the root upload folder, go ahead, only make sure you change the relative current path reference.
